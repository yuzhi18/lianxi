**1.** **地图与初始化**

- 地图规格：20行 × 30列网格（共600格），边界为不可穿越墙壁
- 墙壁占用边缘网格：第0/21行、第0/31列为墙壁区域（用`*`字符渲染）
- 蛇初始化：
  - 外观：`#`（可以自由发挥），和边界区分开
  - 起始位置：蛇头位于(10, 16)，身体向左延伸至(10,15)、(10,14)
  - 初始方向向右（下一步移向17列）
  - 初始长度：3节
  - 初始方向：向右
  - 仅可在内部区域移动：行1-20，列1-30
- 食物初始化：游戏开始时生成1个食物于随机空白格（不与蛇体重叠）

**2.移动机制**

- 自动移动：蛇以固定间隔移动（初始200ms/格）
- 玩家控制：
  - 按键：↑（上）、↓（下）、←（左）、→（右）
  - 禁止180°转向：当前向右时按←无效（其他方向同理）
  - 按键立即生效于下一次移动（两次移动间隔内多次按键 → 仅记录最后一次有效方向（过滤180°转向））
- 速度变化：每吃到1个食物，移动间隔减少10ms（最低100ms）
- 未吃食物：蛇头前进1格 → 身体各节前移 → 移除原蛇尾
- 吃到食物：蛇头前进1格 → 身体前移 → 保留原蛇尾（长度+1）

**3.食物与生长规则**

- 食物生成：
  - 位置：随机空白格（重试≤5次，避免死循环， 触发`GameEndException("地图已满"）`游戏结束，显示“恭喜通关！地图已无空位”）
  - 外观：红色圆形（可以自由发挥），和蛇还有边界样子区分开
- 吃食物效果：
  - 蛇长度+1（新增节附着蛇尾）
  - 得分+10（核心计分依据）
  - 移动速度提升（间隔-10ms）

**4.碰撞检测与游戏结束**

- 结束条件（满足任一即终止）： 蛇头坐标超出边界（行<0 / ≥20 或 列<0 / ≥30） 蛇头与自身任意身体节坐标重合
- 结束动作：
  - 停止移动与计时
  - 显示最终得分、游戏总时长（秒）
  - 提供“重新开始”按钮（重置所有状态）

**5.碰撞检测时序**

1. 蛇头移动至新坐标
2. 优先检测是否吃到食物（坐标重合）→ 是则增长+生成新食物
3. 否则检测碰撞：墙壁/自身 → 是则游戏结束

**6.得分与提交规范**

得分计算：总得分 = 吃到的食物数量 × 10（得分必须为10的正整数倍）

分数范围：理论最大值≈6000（蛇占满地图）0 ≤ score ≤ 10000
